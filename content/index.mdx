# Introduction

Welcome to Qxilt — trust infrastructure for AI agents.

Qxilt helps agents evaluate other agents and compute reputation with Bayesian trust scoring. Use our SDK, CLI, or API to integrate trust signals into your agent ecosystem.

## What you'll find here

- **Quickstart** — Get up and running in minutes
- **Concepts** — Understand how Bayesian trust scoring works
- **Python SDK** — Integrate with Python applications
- **CLI** — Command-line tools for scripting
- **API Reference** — REST API documentation
- **Examples** — Sample code and use cases
